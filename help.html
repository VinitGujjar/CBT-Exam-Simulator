CBT Exam Simulator Help

Prompts, format guide, and tips for best results.

How to Use

  • Paste questions manually, import TXT/CSV, or use Extract from PDF.
  • If text is messy after extraction, use Auto Arrange Text to fix format.
  • Set quiz time, marks per correct answer, and negative marks per wrong answer.
  • Use Exam mode for real test practice. Use Practice mode for instant explanations.
  • Use 🔖 Save button during exam to bookmark important/tricky questions.
  • After quiz results, you can Repeat, Redo Incorrect, practice Not Attempted, or Marked Questions.
  • Use Reset Analysis in the summary pills to start fresh tracking for a new subject/chapter.
  • Questions with year tags like [SSC-JE-2023] or [2021 Shift 1] show a blue badge automatically.

Question Format

Section: Physics
Que 1
What is the SI unit of power? [SSC-JE-2019]
A) Newton
B) Joule
C) Watt
D) Pascal
Answer: C
Explanation: Power is measured in watts. One watt equals one joule per second.

💡 Adding [SSC-JE-2023] or [RRB-JE 2021 Shift 1] in the question shows a blue year badge automatically during exam and result review.

Match the Following Format

✅ Now Supported

For "Match the Following" questions, keep List-I and List-II on their own lines inside the question, and put the four combination options (A, B, C, D) as the last four lines before Answer. The app now detects this automatically.

Section: Physics
Que 1
Match the following:
List I
1. Force
2. Work
3. Power
4. Pressure
List II
i. Watt
ii. Newton
iii. Pascal
iv. Joule
A) 1-ii, 2-iv, 3-i, 4-iii
B) 1-i, 2-ii, 3-iii, 4-iv
C) 1-iii, 2-i, 3-iv, 4-ii
D) 1-iv, 2-iii, 3-ii, 4-i
Answer: A
Explanation: Force is measured in Newton, Work in Joule, Power in Watt, Pressure in Pascal.

⚠️ Important: each combination option (A, B, C, D) must be on its own line, written like A) 1-ii, 2-iv, 3-i, 4-iii. Do not put List I / List II items on the same line as A)/B)/C)/D) options, or the app may misread them.

Best PDF Type

  • Text-based PDFs work best — you should be able to select and copy text from the PDF.
  • Scanned/image PDFs need OCR first (use Adobe Acrobat or free online OCR tools).
  • Two-column PDFs are supported, but tables, formulas, and diagrams may need manual correction.
  • If answers or solutions are at the end, keep them in the same PDF so the app can match by question number.
  • Match-the-following tables in PDFs often need manual review after extraction — verify List I / List II items lined up correctly with their options.

Prompt for ChatGPT / Claude to Create Questions

✅ Best Results 
Create MCQ questions for SSC JE / RRB JE exam practice in exactly this format for CBT Exam Simulator:

Section: [topic name]
Que 1
Question text here? [SSC-JE-2023]
A) option 1
B) option 2
C) option 3
D) option 4
Answer: C
Explanation: Concise, exam-focused explanation of why this answer is correct.

Que 2
...and so on

Strict rules:
- Start every question with Que 1, Que 2, Que 3... (sequential, no gaps).
- Use only four options: A, B, C, D.
- If you know the exam year/shift this question appeared in, add it in the question line as [SSC-JE-2023] or [RRB-JE 2021 Shift 1]. If unknown, leave it out — do not guess or make up years.
- Do not put any blank lines inside one question block.
- Keep Question, A, B, C, D, Answer, and Explanation on consecutive lines.
- Put exactly one blank line only after the Explanation line, before the next question.
- Do not add introductory lines like "Here are the questions", "Sure", or any preamble.
- Answer must be only one letter: A, B, C, or D.
- Explanations must be concise and exam-focused (1–3 lines).
- Do not use bullets, markdown headings, tables, or extra numbering inside questions.
- For "Match the Following" type questions: write "List I" on its own line followed by numbered items (1. 2. 3. 4.), then "List II" on its own line followed by roman-numeral items (i. ii. iii. iv.), and only then the four A)/B)/C)/D) combination options like "A) 1-ii, 2-iv, 3-i, 4-iii" as the last four lines before Answer. Never mix List I/List II items into the same line as an A)/B)/C)/D) option.
